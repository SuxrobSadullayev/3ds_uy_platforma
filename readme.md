### Maqsad
 
O'zbekistondagi ko'chmas mulk bozorini **to'liq raqamlashtirish** va 3D texnologiyalar orqali xaridorlar, sotuvchilar, investorlar va davlat muassasalari uchun **yagona ishonchli platforma** yaratish.
 
Platforma quyidagi operatsiyalarni qamrab oladi:
- 🏠 Uy-joy va tijorat mulklarini **sotish**
- 🔑 Mulklarni **ijaraga berish**
- 🔨 E-auktsion orqali davlat mulklarini **sotish**
- 📈 **Investitsiya** imkoniyatlari
- 🏠➡️💰 **Rent-to-Own** (qo'shimcha modul)

```

│ Rol                  │ Asosiy vazifalari                                   │
```

- ✅ Platforma sozlamalari: komission foizlar, to'lov usullari, konfiguratsiya
- ✅ Real-time analytics dashboard
- ✅ Auditlog — har bir amalni kim, qachon, qaysi IPdan qilganini ko'rish

- ✅ E-auktsion boshqaruvi: boshlash, to'xtatish, g'olibni tasdiqlash
- ✅ Kunlik, haftalik, oylik va yillik hisobotlar
- ✅ Barcha foydalanuvchilarga push/SMS/email yuborish

## ⚙️ Funksional Talablar
 
### 🏢 Mulk E'loni Moduli (Qurilish Kompaniya)
 
| Xususiyat | Tavsif |
|-----------|--------|
| **Mulk turlari** | Kvartira, uy, ofis, do'kon, ombor, qurilayotgan bino, davlat mulki |
| **Media yuklash** | Rasm (JPG/PNG, max 20MB), Video (MP4, max 500MB), 3D model (.glb/.gltf, max 200MB) |
| **Virtual tour** | 360° virtual tour — Matterport yoki Cloudpano integratsiyasi |
| **Narx variantlari** | To'liq narx, ipoteka, bo'lib to'lash, chegirma |
| **Qurilish kuzatuv** | Bosqich nomi, foiz tayyor, kamera oqimi |
| **Rent-to-Own** | ➕ Qo'shimcha modul — mulkni R2O rejimida taklif qilish |
 
### 🛒 Xaridor Kabineti
 
- 🔍 **Qidiruv va filtrlash** — joylashuv, narx, maydon, xona, bino turi
- 🤖 **AI Match tizimi** — parametrlar asosida mos mulklar avtomatik taklif
- 🔲 **3D ko'rish** — brauzerda .glb model, 360° virtual tour, interaktiv floor plan
- 📱 **AR rejimi** — telefon kamerasi orqali mebel va rang simulyatsiyasi (mobil)
- ❤️ **Sevimlilar** — ro'yxat va solishtirish (max 3 ta mulk parallel)
- 💬 **Chat** — sotuvchi bilan matn, rasm, hujjat yuborish
- 📅 **Uchrashuv belgilash** — takvim orqali, tasdiqlash SMS/email
- 💳 **To'lov va shartnoma** — xavfsiz to'lov (escrow), e-imzo
- 🎁 **Referral tizimi** — do'stga tavsiya → bonus ball yoki chegirma
### 📈 Investor Kabineti
 
- 📊 **Portfolio boshqaruv** — sotib olingan mulklar, joriy narx, o'sish foizi
- 🤖 **AI Kuzatuvchi** — narx o'zgarganda avtomatik xabar
- 💰 **ROI kalkulyator** — sotib olish narxi, xarajatlar, bashorat qilingan foyda
- 📈 **Bozor tahlili** — tuman bo'yicha narx grafiklari, 12 oylik dinamika
- 🤝 **Fraksional mulk** *(qo'shimcha)* — boshqa investorlar bilan birgalikda sotib olish
- 🔄 **Qayta sotish** — mulkni platforma orqali sotish, narx belgilash
### 🔨 E-Auktsion Moduli
 
```
📋 Auktsion turlari:
   ├── Ochiq kim oshdi
   ├── Maxfiy taklif (sealed bid)
   └── Vaqt chekli auktsion
 
🔐 Ishtirokchi:       KYC tasdiqlash + garov depozit
⚡ Real-time bidding: Joriy eng yuqori taklif + avtomatik increment
⏱️  Timer countdown:  Auktsion tugash vaqti jonli ekranda
🏆 G'olib:           Avtomatik SMS + email + platforma xabar
📄 Post-auktsion:    Hujjatlar paketi + e-imzo shartnoma
```
 
### 🏠➡️💰 Rent-to-Own Moduli *(Qo'shimcha)*
 
> Ushbu modul asosiy platformaga ulangan **qo'shimcha xususiyat** sifatida ishlaydi. Sotuvchi xohlasa bu rejimni yoqishi mumkin.
 
- 📅 Ijarachi oylik to'lov jadvalini ko'radi va to'laydi
- 🔔 Muddat yakunida uyni sotib olish opsioni avtomatik taklif
- 🔒 **Narx qulflash (price lock)** — boshlang'ich kelishilgan narx o'zgarmaydi
- 📊 To'lov tarixi va qoldiq miqdor real-time dashboardda
- 👤 Sotuvchi: to'lovlar monitoringi, vaqt tugaganda qaror

## 🔲 3D va Vizualizatsiya Talablari
 
### Qo'llab-quvvatlanadigan formatlar
 
| Format | Tavsif | Maks hajm | Brauzerda ko'rish |
|--------|--------|-----------|-------------------|
| `.glb / .gltf` | 3D model (standart) | 200 MB | Three.js / Babylon.js |
| `.obj + .mtl` | 3D model (keski) | 150 MB | Konvertatsiya kerak |
| `360° JPG/MP4` | Virtual tour | 500 MB | Pannellum / CloudPano |
| `.dwg / .dxf` | Arxitektura rejasi | 50 MB | SVG ga konvert |
| `Floor plan PNG` | Xona rejasi | 20 MB | Interaktiv SVG |
 
### 3D Ko'rish interfeysi
 
```
✅ Brauzer ichida yuklanish     — Three.js / Babylon.js, plugin shart emas
✅ Boshqaruv                   — orbit, zoom, reset, to'liq ekran
✅ Yoritish                    — environment lighting, shadow, real-time material
✅ Annotatsiya                 — 3D modelda nuqta: xona nomi + o'lchami
✅ Mobil optimizatsiya         — LOD (Level of Detail)
✅ Yuklanish vaqti maqsadi     — < 10 soniya (100 Mbps internet)
```
